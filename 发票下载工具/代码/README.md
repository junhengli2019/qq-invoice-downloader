# QQ 邮箱发票下载工具

这是一个给普通 Windows 用户使用的小工具。无需输入命令，也不需要安装额外软件包：解压后双击启动文件，在网页中填写信息即可下载发票 PDF。需要 Python 3.7 或以上版本。

## 项目结构

```text
发票下载工具/
├── 代码/
│   ├── download_invoices.py
│   ├── invoice_web_app.py
│   └── README.md
└── 双击此文件开始下载.bat
```

第一次成功保存 PDF 时，程序会在“发票下载工具”目录中自动生成 `downloads` 文件夹。请不要提前创建该文件夹。

## Windows 使用方法

1. 解压整个“发票下载工具”文件夹，不要只复制其中某一个文件。
2. 双击 `双击此文件开始下载.bat`。
3. 黑色命令行窗口保持打开。浏览器会自动访问 `http://127.0.0.1:8765`。
4. 在网页中填写 QQ 邮箱、IMAP 授权码、起始日期和截止日期。
5. 点击“开始下载”，等待页面显示“下载完成”。
6. 到“发票下载工具”目录下的 `downloads` 文件夹查看 PDF。
7. 如果页面提示存在手动链接，请打开 `需要手动下载的发票链接.txt`，在浏览器中逐个处理。

关闭黑色命令行窗口后，本地网页服务会立即停止。下次使用时重新双击启动文件即可。

## 安装 Python 3

如果双击后提示“未检测到 Python 3”：

1. 打开 [Python Windows 下载页面](https://www.python.org/downloads/windows/)；
2. 下载 Python 3 安装程序；
3. 运行安装程序时，务必勾选 `Add python.exe to PATH`；
4. 完成安装后，重新双击 `双击此文件开始下载.bat`。

本工具只使用 Python 自带的标准库，不需要安装 Flask、FastAPI 或其他第三方软件包。

## 获取 QQ 邮箱 IMAP 授权码

IMAP 授权码不是 QQ 密码。请不要把 QQ 登录密码填到工具中。

一般操作步骤：

1. 登录 [QQ 邮箱](https://mail.qq.com/)；
2. 进入邮箱设置中的“账号”或与 POP3/IMAP 服务相关的页面；
3. 开启 IMAP/SMTP 服务；
4. 按页面提示完成身份验证并生成授权码；
5. 将生成的授权码填入工具网页的“IMAP 授权码”字段。

QQ 邮箱页面可能调整菜单名称，请以 QQ 邮箱当前显示的说明为准。授权码应像密码一样保管，不要通过聊天、邮件或截图发给他人。

## 下载规则

- 只扫描 QQ 邮箱收件箱 `INBOX`。
- 日期范围包含起始日期和截止日期当天。
- 只有主题、发件人或正文中包含“发票”的邮件才会处理。
- 邮件附件只保存 PDF；XML、OFD、ZIP、CSV 等附件会跳过。
- 所有 PDF 都保存在同一个 `downloads` 文件夹，不按月份分目录。
- 文件名尽量包含邮件日期、发件人和原始附件名，并会自动清理 Windows 不允许的字符。
- 重复运行时，已有同名 PDF 会跳过，不会覆盖，也不会生成重复副本。
- 正文中的发票下载链接会先尝试自动下载。
- 链接直接返回 PDF 时会保存；链接返回 ZIP 时只提取其中的 PDF，不保存 ZIP 或其他文件。
- 登录页、扫码页、短信验证页、过期链接或其他无法自动处理的页面，会记录到 `需要手动下载的发票链接.txt`。
- `downloads` 和手动链接文件都按实际需要自动生成。

## 隐私与安全

- 工具只监听本机地址 `127.0.0.1`，其他电脑不能直接访问。
- QQ 邮箱和 IMAP 授权码只用于本次运行，不写入配置文件或状态文件。
- 页面状态和运行输出不会显示 IMAP 授权码。
- 工具不会创建 `.env` 或 `download_state.json`。
- 下载链接可能由第三方开票平台提供。无法直接得到 PDF 的链接会留给用户手动打开，不会让整个任务中断。

## 哪些文件不要发给客户

如果需要把这个工具转发给其他人，请先确认没有包含以下内容：

- `downloads` 文件夹及其中的个人发票；
- `需要手动下载的发票链接.txt`，其中可能包含带临时凭证的下载链接；
- `.env`、`download_state.json` 或任何包含邮箱、密码、授权码的文件；
- `__pycache__`、`.DS_Store`、`__MACOSX`、任何以 `._` 开头的系统文件；
- 运行时生成的其他个人数据。

最安全的做法是只发送最初下载或解压出来、从未填写个人邮箱运行过的干净项目文件夹。

## 常见问题

### 浏览器没有自动打开

确认黑色命令行窗口仍然显示“QQ 邮箱发票下载工具已启动”，然后手动在浏览器地址栏输入：`http://127.0.0.1:8765`。

### 提示登录失败

请依次确认：邮箱地址正确、填写的是 IMAP 授权码而不是 QQ 密码、QQ 邮箱已开启 IMAP 服务。

### 提示端口被占用

关闭之前打开的工具黑色窗口，再重新双击启动文件。

### 没有生成 downloads 文件夹

`downloads` 只会在第一次成功保存 PDF 时自动生成。如果没有匹配的发票邮件、PDF 已存在或邮件只提供手动下载链接，就不会创建空文件夹。
